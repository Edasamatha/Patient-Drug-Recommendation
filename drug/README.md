# Patient Condition-Aware Drug Recommendation + Clinical Review Sentiment (Flask + Streamlit)

Advanced end-to-end project with:
- Deterministic, condition-aware drug recommendation (rule-based clinical safety scoring)
- Clinical review sentiment analysis via Hugging Face Inference API
- Flask backend API
- Streamlit frontend
- No local model downloads

## Why this reduces hallucination risk
- Drug recommendation is **not generated by an LLM**.
- Recommendations come from a curated local knowledge base (`api/data/drug_knowledge.json`) with explicit contraindication/caution rules.
- Hugging Face API is used only for sentiment classification.

## Setup

```bash
cd /home/rohini/Downloads/CIET/drug
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
```

Update `.env` with your key:

```env
HF_API_KEY=your_huggingface_api_key_here
HF_SENTIMENT_MODEL=siebert/sentiment-roberta-large-english
HF_SENTIMENT_FALLBACK_MODELS=cardiffnlp/twitter-roberta-base-sentiment-latest,distilbert/distilbert-base-uncased-finetuned-sst-2-english
FLASK_HOST=0.0.0.0
FLASK_PORT=5000
```

## Run backend (Flask)

```bash
source .venv/bin/activate
python api/app.py
```

Health check:

```bash
curl http://127.0.0.1:5000/health
```

## Run frontend (Streamlit)

```bash
source .venv/bin/activate
streamlit run app.py
```

When Streamlit starts, it automatically attempts to start Flask backend using:

```bash
python -m api.app
```

If you want manual control, set:

```env
BACKEND_AUTOSTART=false
```

## API usage

### 1) Drug recommendation

```bash
curl -X POST http://127.0.0.1:5000/recommend \
  -H "Content-Type: application/json" \
  -d '{
    "condition": "hypertension",
    "age": 56,
    "pregnant": false,
    "allergies": ["penicillin"],
    "comorbidities": ["chronic_kidney_disease"],
    "current_medications": ["Aspirin"],
    "renal_impairment": true,
    "liver_impairment": false
  }'
```

### 2) Sentiment analysis

```bash
curl -X POST http://127.0.0.1:5000/sentiment \
  -H "Content-Type: application/json" \
  -d '{
    "reviews": [
      "Patient tolerated metformin well with improved glucose.",
      "Experienced significant nausea and fatigue."
    ],
    "model": "siebert/sentiment-roberta-large-english"
  }'
```

### 3) Dynamic recommendation from text + uploaded report

```bash
curl -X POST http://127.0.0.1:5000/recommend_dynamic \
  -F 'description=56 year old with high blood pressure, CKD stage 3, eGFR: 48, creatinine: 1.8' \
  -F 'reports=@sample_report.txt'
```

Supported uploaded report formats: `.txt`, `.csv`, `.json`, `.pdf`.
